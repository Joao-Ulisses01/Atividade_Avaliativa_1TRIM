Sobre o h1

o titulo principal de uma pagina web, sendo o elemento mais importante na estruturação de conteudo HTML

Sobre o h2

usada para definir o segundo nivel de título, funcionando como um subtitulo para organizar seçoes e subseçoes.

Sobre o h3

Um elemento de cabeçalho de terceiro nivel, utilizado para definir subtitulos dentro de um documento.

Sobre o h4

Representa o quarto nivel de titulo, em uma hierarquia de documentos, sendo usada para subtitulos especificos.

Sobre o h5

Utilizado para definir sub-sub-sub-subtitulos

Sobre o h6

Nivel mais baixo e menos importante na hierarquia de titulos
